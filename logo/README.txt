
dgMic Variant C Icon Pack
=========================

This archive contains the Variant C microphone icons derived from the dgpays logo. Variant C features a rounded rectangular head with horizontal slots.

Files
-----
- **dgMicRect-1024.png** ... **dgMicRect-16.png** – PNG icons from 1024px down to 16px.
- **tray-light-16.png**, **tray-light-20.png**, **tray-light-24.png** – White tray icons for dark system themes.
- **tray-dark-16.png**, **tray-dark-20.png**, **tray-dark-24.png** – Black tray icons for light system themes.
- **dgMicRect.ico** – Windows ICO file containing multiple sizes.

Usage instructions are similar to those provided previously: set the app icon via `BrowserWindow`'s `icon` property, use the ICO on Windows, and swap tray icons based on dark/light mode.
